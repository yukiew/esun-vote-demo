1. 系統概述
我們的投票系統允許您創建、刪除和參與投票活動。

2. 登錄與用戶模式切換
首次進入系統，您將進入使用者模式。要切換到管理員模式，請點擊畫面右上角的「使用者模式」按鈕。同樣地，您可以在管理員模式下切換回使用者模式。

3. 管理投票項目
在管理員模式下，您可以進行以下操作：

a.新增投票項目
在輸入框中輸入您要新增的投票項目，按下鍵盤的「Enter」鍵。
新的投票項目將會被創建並保存。

b.刪除不再需要的投票項目
滑鼠放在投票項目然後點擊右下「刪除」。

4. 投票流程
在使用者模式下，您可以進行投，
在投票項目列表中選擇您要參與的投票(可多選)，
選擇您支持的選項，然後點擊右下「投票」按鈕。

5. 查看票數
不論是使用者模式還是管理員模式，您都可以查看投票票數
